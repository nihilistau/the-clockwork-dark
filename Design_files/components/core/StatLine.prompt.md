One-sentence: A single ledger row — smallcaps label + monospace tabular value with a hairline rule — stacked to build the character sheet.

```jsx
<StatLine label="HP" value="14/18" />
<StatLine label="Stamina" value="6" />
<StatLine label="Gold" value="0.42" accent />
```

Values render tabular-nums in mono so columns align. Keep labels one word. `accent` tints the value brass.
