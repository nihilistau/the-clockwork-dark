# UI Kit — The Clockwork Dark Scene

An interactive recreation of the game's single playable scene, faithful to `content/scenes/clockwork/templates/clockwork.html` and `static/js/clockwork.js`.

## Files
- `index.html` — mounts the scene, loads the DS bundle + `styles.css`.
- `GameScene.jsx` — the whole interactive scene: archetype start screen, three-column journal frame (assistant · narrative · sheet), a fake turn loop (choice → dice toast → narration → stat update), and a phase switcher in the footer that re-tints the UI through dormant → stirring → spreading → consuming.

## What it demonstrates
- The fixed three-column frame (`200px · flex · 220px`) under chrome header (WorldClock) and footer (day · time · weather · phase).
- Composition of every DS component: `Button`, `ChoiceChip`, `Badge`, `StatLine`, `AssistantBubble`, `DiceToast`, `ScenePanel`, `WorldClock`.
- The narrative log (serif, paper grain), player-choice echo with brass edge, non-blocking centered dice toast.
- Phase themes driving the whole surface from one `[data-phase]` attribute.

## Not included (canon / scope)
- Real scene art — the visual is a painterly placeholder; production art is generated by ComfyUI at runtime.
- Cutscene letterbox, push-to-talk waveform, and combat sheet are documented in the brief but omitted here to keep the kit focused on the core turn loop.
- No real Socket.IO / backend; the turn loop is faked client-side.
